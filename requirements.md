# Project Requirements

## Project Name:
Smart Risk Analyzer & Reminder System

## Objective:
Develop a lightweight, low-cost system that collects user data, performs rule-based risk analysis, generates suggestions, and sends reminders, suitable for hackathon submission.

## Core Features:
1. **User Data Entry** – Simple form for entering health or project data.
2. **Data Storage** – Store user data in a free-tier database (Firebase / SQLite).
3. **Risk Analysis Module** – Rule-based evaluation of user input.
4. **Suggestions Generator** – Provides actionable advice based on analysis.
5. **Reminder & Notification System** – Sends reminders via email, WhatsApp, or in-app notifications.
6. **Dashboard** – Minimal view of data, analysis results, suggestions, and reminders.

## Tech Stack:
- Frontend: Flutter / React Native / HTML + CSS + JavaScript
- Backend: Python Flask / Node.js
- Database: Firebase / SQLite
- Notifications: Email / WhatsApp API (free tier)
- Deployment: Render / Vercel / Firebase free tier

## Estimated Low-Budget Cost (INR):
- Frontend UI: ₹5,000 – ₹10,000
- Backend + Logic: ₹2,000 – ₹5,000
- Database: ₹0 – ₹2,000
- Risk Analysis: ₹0 – ₹2,000
- Notifications / Reminders: ₹0 – ₹1,000
- Deployment & Hosting: ₹0 – ₹1,000/year

**Total Estimated Cost:** ₹7,000 – ₹20,000
